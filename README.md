## ūüĆć What Is This?

A **production-ready, end-to-end watershed analysis pipeline** for the **Pravara River Basin** ‚ÄĒ a Godavari sub-basin in Maharashtra, India. Built entirely in Python and designed to run on Google Colab with a single zip file upload.

The pipeline covers **18 sequential sections** across three thematic modules:

| Module | Sections | What It Does |
|--------|----------|-------------|
| ūüĒĶ **Core Morphometry** | 0 ‚Äď 9 | 62+ basin shape (including Lemniscate & Basin Shape indices), drainage, relief params, maps, PCA |
| ūüü† **Geomorphic & Tectonic** | 10 ‚Äď 13 | Tectonic activity indices, SL gradient, TWI/SPI rasters, flood hazard composite |
| ūüüĘ **Hydrology & SWC** | 14 ‚Äď 18 | SCS-CN runoff, RUSLE erosion, conservation planning, unit hydrographs, channel hydraulics |
