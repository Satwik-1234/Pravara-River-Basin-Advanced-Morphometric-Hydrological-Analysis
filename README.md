## рҹҢҚ What Is This?

A **production-ready, end-to-end watershed analysis pipeline** for the **Pravara River Basin** вҖ” a Godavari sub-basin in Maharashtra, India. Built entirely in Python and designed to run on Google Colab with a single zip file upload.

The pipeline covers **18 sequential sections** across three thematic modules:

| Module | Sections | What It Does |
|--------|----------|-------------|
| рҹ”ө **Core Morphometry** | 0 вҖ“ 9 | 62+ basin shape (including Lemniscate & Basin Shape indices), drainage, relief params, maps, PCA |
| рҹҹ  **Geomorphic & Tectonic** | 10 вҖ“ 13 | Tectonic activity indices, SL gradient, TWI/SPI rasters, flood hazard composite |
| рҹҹў **Hydrology & SWC** | 14 вҖ“ 18 | SCS-CN runoff, RUSLE erosion, conservation planning, unit hydrographs, channel hydraulics |
